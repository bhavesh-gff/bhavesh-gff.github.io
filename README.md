A modern responsive personal portfolio website built using HTML, CSS, JavaScript, and Tailwind CSS.
This portfolio showcases my skills, projects, resume, experience, and contact information in a clean and professional layout.

 Features

- Responsive modern UI design
- Home, About, Resume, and Projects sections
- Skills and experience timeline
- Project showcase cards
- Contact form
- Social media integration
- Smooth animations and transitions
- Mobile-friendly design

Technologies Used

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Remix Icons
- Google Fonts (Inter)

How to Run

### Option 1: Open Directly

Simply open `index.html` in your browser.

### Option 2: Using VS Code Live Server

1. Open project folder in VS Code
2. Install Live Server extension
3. Right click on `index.html`
4. Click **Open with Live Server**
